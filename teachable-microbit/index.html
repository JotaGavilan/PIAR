ℹ️ Ajuda

Teachable Microbit et permet utilitzar models d'intel·ligència artificial creats amb Google Teachable Machine i enviar les prediccions a una micro:bit per Bluetooth en temps real.

Com funciona:

  1. Crea un model a Teachable Machine
  2. Exporta'l i puja'l (rebràs un codi com WIQv36jWP)
  3. A Configuració, enganxa el codi i carrega el model
  4. Connecta la micro:bit per Bluetooth
  5. Les prediccions s'enviaran automàticament!

Tipus de models compatibles:

  • 🖼️ Imatge — detecta objectes, gestos, posicions davant la càmera
  • 🎤 Audio — reconeix sons, paraules, música (requereix micròfon)
  • 🧍 Postura — detecta posicions del cos sencer

Format de les dades enviades:

  • AMB probabilitats: gat:87;gos:10;ocell:3
  • SENSE probabilitats: gat;gos;ocell
  • Quan no hi ha predicció segura, s'envia 0 una vegada
